# HTML

## 文字样式

| 标记                  | 显示效果      |
| --------------------- | ------------- |
| `<b></b>`             | <b>加粗</b>   |
| `<i></i>`             | <i>斜体</i>   |
| `<u></u>`             | <u>下划线</u> |
| `<s></s>`             | 删除线        |
| `<big></big>`         | 放大          |
| `<small></small>`     | 缩小          |
| `<strong></strong>`   | 加强          |
| `<em></em>`           | 强调          |
| `<address></address>` | 显示网址      |
| `<code></code>`       | 代码块        |
